# Furina Landing Page Project

## Project Overview
This website is a landing page project created specifically for an academic assignment. The theme of the website is based on the character Furina from the game Genshin Impact. The project demonstrates the integration of modern web technologies, including 3D model rendering, interactive background effects, and responsive design.

## Features
- Interactive 3D model viewer.
- Dynamic water ripple effect on mouse interaction.
- Animated star particle background overlay.
- Responsive layout with glassmorphism aesthetics.
- Multimedia integration including animated image previews.

## Technical Details
The project is built using the following technologies:
- HTML5 and CSS3
- JavaScript
- Model-viewer for 3D rendering
- jQuery Ripples for water interaction
- tsParticles for background animations
